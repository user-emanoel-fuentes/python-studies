# python-studies

## 🎯 Objetivo do Repositório

Este repositório é um ambiente dedicado ao estudo e prática da linguagem de programação **Python**. Aqui, coleciono exemplos de código, scripts, exercícios e pequenos projetos que me ajudam a consolidar os fundamentos da linguagem, explorar sua vasta biblioteca padrão, e aplicar conceitos de programação orientada a objetos, tratamento de dados e automação.

## ��️ Mapa de Conteúdo

*   **Fundamentos da Linguagem:** Variáveis, tipos de dados, operadores, estruturas de controle.
*   **Estruturas de Dados:** Listas, tuplas, dicionários, conjuntos.
*   **Funções e Módulos:** Criação e organização de código reutilizável.
*   **Programação Orientada a Objetos (POO):** Classes, objetos, herança, polimorfismo.
*   **Manipulação de Arquivos:** Leitura e escrita em diferentes formatos (texto, CSV, JSON).
*   **Tratamento de Exceções:** Gerenciamento de erros de forma robusta.
*   **Pequenos Scripts e Automações:** Exemplos de uso prático da linguagem.

## �� Descrição dos Conteúdos

### Fundamentos da Linguagem
Contém exemplos básicos da sintaxe Python, operadores, loops (`for`, `while`), condicionais (`if/elif/else`) e outros conceitos essenciais.

### Estruturas de Dados
Demonstrações e exercícios práticos com as estruturas de dados built-in do Python, mostrando suas particularidades e melhores usos.

### Funções e Módulos
Exemplos de como definir funções, passar argumentos, usar módulos e pacotes para organizar projetos Python.

### Programação Orientada a Objetos (POO)
Classes, objetos, métodos, atributos, encapsulamento, herança e polimorfismo explicados com exemplos práticos.

### Manipulação de Arquivos
Scripts que mostram como ler e escrever dados em arquivos de texto, CSV, JSON e outros formatos comuns.

### Tratamento de Exceções
Exemplos de como usar blocos `try-except-finally` para lidar com erros e tornar os programas mais resilientes.

### Pequenos Scripts e Automações
Uma coleção de scripts úteis para tarefas cotidianas, como processamento de texto, renomeação de arquivos, web scraping básico, etc.
